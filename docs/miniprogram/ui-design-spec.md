## 1. 设计原则

### 1.1 核心设计理念

| 原则 | 说明 |
|---|---|
| **极简主义** | 减少视觉噪音，让内容成为主角 |
| **科技感** | 深色主题 + 蓝色点缀，契合 AI/学术工具定位 |
| **高效阅读** | 信息层级清晰，三段式摘要一目了然 |
| **一致性** | 统一的组件、间距、色彩，降低学习成本 |

### 1.2 设计价值观

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   "让用户在碎片时间里，快速获取论文核心价值"                   │
│                                                             │
│   • 3秒内理解论文主题                                        │
│   • 30秒内获取 AI 摘要精华                                   │
│   • 1分钟内决定是否深入阅读                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 2. 设计系统

### 2.1 色彩系统

#### 主色板

```
┌────────────────────────────────────────────────────────────────┐
│  深海黑（背景）    │  ████  │  #0d1117  │  页面背景、输入框   │
│  暗灰（卡片）      │  ████  │  #161b22  │  卡片、区块背景     │
│  边框灰           │  ████  │  #30363d  │  边框、分割线       │
│  智能蓝（主色）    │  ████  │  #58a6ff  │  主按钮、激活状态   │
│  深蓝（标签背景）  │  ████  │  #1f4168  │  分类标签背景       │
│  成功绿           │  ████  │  #3fb950  │  摘要"已生成"状态   │
└────────────────────────────────────────────────────────────────┘
```

#### 文字色彩层级

| 层级 | Hex | 用途 |
|---|---|---|
| **Primary** | `#e6edf3` | 标题、重要正文 |
| **Secondary** | `#c9d1d9` | 摘要正文、次要内容 |
| **Muted** | `#8b949e` | 作者名、标签、提示文字、时间 |

#### 功能色

| 状态 | Hex | 使用场景 |
|---|---|---|
| 成功 | `#3fb950` | 摘要生成完成、操作成功 |
| 警告 | `#d29922` | 加载中、等待状态 |
| 错误 | `#f85149` | 操作失败、网络错误 |
| 信息 | `#58a6ff` | 链接、可点击元素 |

### 2.2 字体系统

#### 字号规范

```
┌─────────────────────────────────────────────────────────────┐
│  页面标题（导航栏）    17px  / 500  / Primary                 │
│  论文标题（详情）      16px  / 600  / Primary                 │
│  论文标题（卡片）      14px  / 400  / Primary                 │
│  摘要标题（What/How）  14px  / 500  / Primary                 │
│  正文 / 摘要内容       14px  / 400  / Secondary               │
│  作者名 / 标签         12px  / 400  / Muted                   │
│  分类标签 / 状态徽章   11px  / 400  / Accent                  │
│  辅助说明 / 免责声明   11px  / 400  / Muted                   │
└─────────────────────────────────────────────────────────────┘
```

#### 行高规范

| 类型 | 行高 |
|---|---|
| 标题 | 1.4 |
| 正文 | 1.6 |
| 摘要 | 1.7（提升可读性） |

### 2.3 间距系统

#### 基础单位

```
基础单位: 4px

间距阶梯:
  4px   = 1x (微间距)
  8px   = 2x (元素内间距)
  12px  = 3x (卡片间距)
  16px  = 4x (区块间距)
  24px  = 6x (页面边距 - 大屏)
  32px  = 8x (主要区块分隔)
```

#### 页面内边距

| 页面类型 | 内边距 |
|---|---|
| 列表页 | 左右 12px |
| 详情页 | 左右 16px |
| 设置页 | 左右 16px |

### 2.4 圆角系统

```
┌─────────────────────────────────────────────┐
│  大圆角    12px  │  卡片、弹窗              │
│  中圆角     8px  │  输入框、次级按钮        │
│  小圆角     6px  │  标签、徽章              │
│  全圆角   999px  │  圆形图标按钮            │
└─────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────┐
│  ┌─────────────────────────────────────────────────────┐    │
│  │  ┌──────┐ ┌──────┐                                   │    │
│  │  │ cs.AI│ │ cs.CL│                      ← 分类标签   │    │
│  │  └──────┘ └──────┘                                   │    │
│  │                                                       │    │
│  │  Attention Is All You Need                   ← 标题  │    │
│  │                                                       │    │
│  │  Vaswani 等                                  ← 作者  │    │
│  │                                                       │    │
│  │  ┌─────────────────────────────────────────────┐     │    │
│  │  │ ● AI 摘要已生成                              │ ← 状态│    │
│  │  └─────────────────────────────────────────────┘     │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                             │
│  尺寸: 宽度自适应(左右各12px边距) x 高度约120px              │
│  背景: #161b22                                              │
│  圆角: 12px                                                 │
│  内边距: 16px                                               │

└─────────────────────────────────────────────────────────────┘
```

#### 状态变体

| 状态 | 状态标签样式 | 说明 |
|---|---|---|
| `done` | 绿色圆点 + "AI 摘要已生成" | 摘要已完成 |
| `processing` | 橙色圆点 + "AI 摘要生成中..." | 正在生成 |
| `pending` | 灰色圆点 + "等待生成摘要" | 排队中 |
| `failed` | 红色圆点 + "生成失败" | 需重试 |

#### 交互状态

| 状态 | 视觉反馈 |
|---|---|
| 默认 | 背景 `#161b22` |
| 按下 | 背景 `#21262d`，透明度 0.8 |

| iPhone SE | 320px - 375px | 标准布局，边距 12px |
| iPhone 标准版 | 375px - 414px | 标准布局，边距 12-16px |
| iPhone Plus/Max | 414px - 428px | 内容居中，最大宽度 500px |
| iPad | 768px+ | 左右分栏或内容居中 |

### 7.2 安全区域适配


## 9. 动效规范

### 9.1 动效原则

| 原则 | 说明 |
|---|---|
| **自然** | 遵循物理规律，使用 ease-out 曲线 |
| **快速** | 交互反馈不超过 100ms，转场不超过 300ms |
| **有意义** | 动效服务于功能，不单纯装饰 |

### 9.2 缓动曲线

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  默认缓动: ease-out                                         │
│    cubic-bezier(0, 0, 0.2, 1)                              │
│                                                             │
│  进入动效: ease-out                                         │
│    cubic-bezier(0, 0, 0.2, 1)                              │
│                                                             │
│  退出动效: ease-in                                          │
│    cubic-bezier(0.4, 0, 1, 1)                              │
│                                                             │
│  弹性动效: ease-out-back                                    │
│    cubic-bezier(0.34, 1.56, 0.64, 1)                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 9.3 常用动效时长

| 动效类型 | 时长 |
|---|---|
| 按钮按下反馈 | 100ms |
| Toast 出现/消失 | 200ms |
| 页面转场 | 300ms |
| 下拉刷新动画 | 1000ms (循环) |
| 加载旋转动画 | 1000ms (循环) |
| 骨架屏扫光 | 1500ms (循环) |
